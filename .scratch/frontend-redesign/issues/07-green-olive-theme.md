## 07 - Green/olive light theme

**Status:** ready-for-agent

### Description

Replace the current purple/dark theme with a light green/olive gradient theme. Update CSS variables in `index.css` and `App.css`. Remove the `prefers-color-scheme` dark mode override.

### Acceptance Criteria

- Background is a green/olive gradient (`#f5f7f0` to `#e8f0dc`).
- Accent color is green (`#6b8f3c`).
- Text is dark-on-light for readability.
- No dark mode auto-switching.
- Completed todos have muted styling.

### Testing

- Visual/manual verification of theme across all components.
