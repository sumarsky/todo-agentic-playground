## 06 - Add lucide-react icons with tooltips

**Status:** ready-for-agent

### Description

Install `lucide-react` and replace text buttons with icon buttons that have tooltips on hover. Icons needed: `Plus` (add), `Trash2` (delete), `Pencil` (edit), `Search` (search input), `Filter` (filter toggle).

### Acceptance Criteria

- All action buttons use icons instead of text.
- Hovering over an icon shows a tooltip with the action name.
- `aria-label` attributes are preserved for accessibility.

### Testing

- Visual/manual verification of icons and tooltips.
- Accessibility test: aria-labels present on all icon buttons.
