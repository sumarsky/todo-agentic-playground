# Issue: Frontend - React scaffold + TodoContext

## What to build

Set up React app with Tailwind CSS and establish the `TodoContext` state shape. This is the foundation for all frontend work.

**End-to-end**: Create React app (vite or CRA), configure Tailwind CSS, define `TodoContext` with state shape (todos array, loading, error). Render a simple placeholder component.

## Acceptance criteria

- [ ] React app created and runs on `http://localhost:3000`
- [ ] Tailwind CSS configured and working
- [ ] `TodoContext` defined with state shape: `{ todos: [], loading: false, error: null }`
- [ ] Context provider wraps App component
- [ ] Basic test setup (Jest + React Testing Library)

## Blocked by

None - can start immediately
