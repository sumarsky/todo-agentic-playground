# Agent Skills

This document describes the configuration for engineering skills used in this project.

## Agent skills

### Issue tracker

Issues and PRDs live as markdown files in `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage roles map to status fields in issue markdown files. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo with `CONTEXT.md` and `docs/adr/` at the root. See `docs/agents/domain.md`.
